# NotesApp

A beginner-friendly demonstrations on the application of **Firebase** products: 
* **Firebase Authentication**: Register, Login, Logout, Forget Password
* **Firebase Firestore**: Create, Read, Update, Delete (notes).

## Prerequisites:
### Step 1:
To use this project, firstly, follow the instructions on creating the project and app of firebase through either of the documentations:
1. [Canva Slides](https://www.canva.com/design/editor/shell?designId=DAHErWJBaIw&extension=u4JZL6CJAz33K3jD_XbUBQ&mode=edit)
2. [Project Report](https://github.com/user-attachments/files/26879677/Group.1.-.Notes.App.using.Firebase.Firestore.and.MVVM.2.pdf)

### Step 2:
Download **google-services.json** and place it on the level of app directory. After that you could start testing the app, with the features listed above.
